Tytuł testu: Zbiorczy test sprawdzenia pola input NIP na stronie empik.com/empikplace/wniosek

Cel

Sprawdzić, czy pole input NIP na stronie działa poprawnie.

Kroki

  1. Otwórz stronę https://www.empik.com/empikplace/wniosek.
  2. Zaakceptuj cookie.
  3. Znajdź pole input NIP na stronie.
  4. Wpisz litery w polu input NIP i kliknij "Pobierz dane".
  5. Wpisz 10 znaków int w polu input NIP i kliknij "Pobierz dane".
  6. Wpisz 1000 znaków int w polu input NIP i kliknij "Pobierz dane".
  7. Wpisz 1 znak w polu input NIP i kliknij "Pobierz dane".
  8. Wpisz znaki specjalne w polu input NIP i kliknij "Pobierz dane".

Oczekiwane wyniki

  1. Alert powinien się wyświetlić, jeśli wpisane są litery.
  2. Pole input NIP powinno przyjąć 10 znaków.
  3. Pole input NIP nie powinno przyjąć 1000 znaków.
  4. Pole input NIP nie powinno przyjąć znaków specjalnych.
  5. Alert powinien się wyświetlić, jeśli wpisany jest 1 znak.

Kryteria akceptacji

Test jest uznany za zakończony sukcesem, jeżeli wszystkie oczekiwane wyniki są spełnione.
